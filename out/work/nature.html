nhm nature

Drupal 7 Site with custom Blog frontend code. I worked with Hello Design to build the Nature Map using Google Maps API and Inaturalist API to showcase urban nature in Los Angeles. I also built real-time leader board for another urban nature initiative which is a global project, the city nature challenge. The leader board was used as a tool to update local participants on who was leading in observations.
